**FRAMED**


**Overview**

Framed is a first-person detective puzzle game, focusing on observation and deduction.


**Gameplay**

Explore the crime scene.

Collect evidence using the camera.

Record thoughts in the notebook.

Fill out a report with your final conclusion.


**Controls**

WASD - move,
Mouse -look around,
Scroll - zoom in/out,
Esc - return to main menu,
C - open/close camera,
Space - take a picture,
N - open/close notebook,
R - fill out a report


**System Requirements**

Windows


**Installation**

Download the **Framed_Build.zip** file

Extract the folder and open it

Run CAPSTONE.exe

(Backup link to file: https://drive.google.com/file/d/1rRNqX0NHqAYP7krhhv_ymQFvYOdu0raR/view?usp=drive_link)


**Saving**

Screenshots are stored in:
AppData/LocalLow/DefaultCompany/CAPSTONE/Screenshots/

Notes are saved as player_notes.txt in:
AppData/LocalLow/DefaultCompany/CAPSTONE/

These are deleted automatically when the game ends.


**Known Issues**

Occasional cursor-lock inconsistencies when transitioning between menus.
